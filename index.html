MAZE GAME

Creating a game in 3D using raycasting, Raycast 3D Game represents a significant milestone in the realm of game development. By harnessing the power of raycasting, players are transported into a dynamic world where exploration knows no bounds. This ALX/Holberton School foundation project not only showcases the technical prowess of modern game development techniques but also serves as a learning platform for aspiring developers to hone their skills and unleash their creativity

GAME CONTROLS


Arrow Keys -- Player Directions
R -- Refresh
C -- Zoom Player view
[ -- Zoom Out in Game
] -- Zoom In in Game
F -- Change Views (Normal view to Panoramic view and vice versa)
T -- Change Texture of walls and square columns
